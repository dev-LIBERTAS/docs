# ワールド説明

# ワールド全体図

  1. スポーン地点
    手前の感圧版を踏むと埋め立て用のダイヤブロックが手に入ります。

  2. 金リンゴボタン
    ボタンを押すと、金リンゴが一つ手に入ります。

  3. アイテムボタン
    ボタンを押すと、基礎ツール類が手に入ります。

  4. 晴れボタン
    レバーを引くと天候が晴れになります。

  5. 妨害終了ボタン
    ○○祭りが来た時にレッドストーントーチを破壊すると、鶏のスポーンが止まります。


    水族館が来た時にレバーを引くと、水が止まります。　　

  6. コマンド
    ワールドの操作や、難易度の設定ができます。
